How to Connect Looker Studio to Your Funnel Data
Step 1: Create New Report
  1. Go to Looker Studio
  2. Click "Create" → "Report"
  3. Choose "BigQuery" as your data source
Step 2: Connect to BigQuery
  1. Select your project: {{ config.project_id }}
  2. Choose dataset: {{ config.destination_dataset }}
  3. Select table: {{ config.funnel_name }}_metrics
Step 3: Create Funnel Chart
  1. Add a "Funnel chart" to your report
  2. Set Dimension: screen_name
  3. Set Metric: starters_count
  4. Add another metric: completes_count
Step 4: Add Conversion Rate
  1. Create a "Table" chart
  2. Add columns: screen_name, screen_conversion_rate, drop_off_rate
  3. Format conversion rate as percentage
